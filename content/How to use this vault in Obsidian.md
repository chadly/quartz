These are useful hotkeys for creating and editing notes in Obsidian:

| Hotkey             | Action                                |
| ------------------ | ------------------------------------- |
| `Ctrl + N`         | New seed note                         |
| `Ctrl + Q`         | Quick Add Menu                        |

| `Ctrl + E`         | toggle note edit/view mode            |
